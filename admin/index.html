Quick Guide

Welcome to the Coral Sea Training admin panel. Here you can manage your website content:

  • Courses: Update course prices, descriptions, and features
  • Testimonials: Add, edit, or remove customer testimonials
  • Contact Info: Update phone numbers, email, and addresses
  • Settings: Change admin password and export/import data
Note: Changes made here are stored locally. Use the Export feature in Settings to save your changes and apply them to the live site.
